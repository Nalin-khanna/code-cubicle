
# LeafLens

AI Powered Forest Management


## Prerequisites

- node
- npm or yarn
- python
- pip
## Run Locally

Clone the project

```bash
  git clone https://github.com/suj007web/code-cubicle.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the frontend server:

```bash
  npm run dev
```

Start the frontend server:

```bash
    python server.py
